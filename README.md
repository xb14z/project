## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- MongoDB 6+

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd delivery-system
```

2. Install dependencies
```bash
npm install
```

3. Setup environment variables
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. Seed sample data
```bash
npm run seed
```

5. Start the server
```bash
# Development
npm run dev

# Production
npm start
```
